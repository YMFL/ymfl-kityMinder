```
npm i ymfl-kityminder
```

## main.js
```
import 'ymfl-kityminder/dist/index.css'
import kityMinder from 'ymfl-kityminder'

Vue.use(kityMinder)

```

## 组件中使用

```
<kityMinder :importJson="importJson" @minder="minderHandle" />
```
